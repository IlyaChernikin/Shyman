Загадка №1

2x2

Здесь (game.belstu.by/light_webolomka/ - nosource) находятся 10 уровней для решения. Для прохождения потребуется соображалка.
Используйте только то, что видите на экране.
Немного о правилах:
  • Для решения задачи необходимо отгадать правильный URL адрес.
  • Каждая следующая задача добавляет к URL адресу очередной путь.
    Т.е. адрес будет выглядеть так http://game.belstu.by/webolomka/2/ответ1/ответ2/...
  • В написании пути могут использоваться латинские буквы (регистронезависимы), цифры, и символы: .,-+()/$:;_@
  • Не бойтесь ошибаться. На некоторых уровнях есть неправильные страницы с подсказками.
